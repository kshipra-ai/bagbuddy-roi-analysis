# Investor ROI Analysis

This directory contains ROI (Return on Investment) analysis tools and calculations specifically for investors.

## Purpose

Track and analyze the return on investment for investors in the platform, including:
- Portfolio performance
- Investment returns
- Risk assessment
- Growth projections
- Comparative analysis

## Files

- `roi_calculator.py` - Python script for calculating investor ROI metrics
- `metrics.md` - Documentation of key metrics and KPIs
- `sample_data.csv` - Sample data template for analysis
